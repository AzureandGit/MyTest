# My NPM Test

**Create a new directory for your project:**  
```mkdir my-node-app```  
```cd my-node-app```

**Initialize npm**  
```npm init -y```  

**Run the app using the following command:**  
```node index.js```  

**Test the App**    
Open your web browser and go to http://localhost:3000/. You should see a "Hello, World!" message displayed in the browser.
